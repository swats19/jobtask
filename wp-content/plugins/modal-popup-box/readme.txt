== Installation ==

Install Modal Pop Box builder either via the WordPress.org plugin directory or by uploading the files to your server.

After activating Modal Pop Box maker, go to plugin menu.

Click Add Modal Pop Box and write the content in the text box.

Publish the modal pop box builder and copy shortcode from the bottom of the add images box and embed shortcode on any Page/Post/Text Widget.

That's it. You're ready to go!
